Project Overview

The Water Jar Distributor Management System is a desktop-based application developed using C# (.NET) for the frontend and MySQL as the backend database.

The main goal of this project is to automate and simplify the daily operations of a water jar distribution business.

Problem It Solves

  • Order tracking issues
  • Errors in customer records
  • Difficulty in managing jar returns and payments

Key Features

Customer Management

  • Add, update, and delete customer details
  • Store address, contact number, and jar requirements

Order Management

  • Record daily water jar orders
  • Track delivered and pending orders

Jar Tracking

  • Maintain count of jars issued and returned
  • Reduce loss or misplacement of jars

Billing & Payment Tracking

  • Generate customer bills
  • Track paid and pending amounts

Technologies Used

  • Frontend: C# (.NET – Windows Forms)
  • Backend: MySQL
  • Database Connectivity: ADO.NET

Outcome

  • Improved accuracy in records
  • Better time efficiency
  • Improved business reporting
